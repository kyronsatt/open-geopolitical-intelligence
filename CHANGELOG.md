## Version History

### v1.0.0 (2024-03-04)

#### Features
- Full conflict analysis dashboard
- Real-time LLM integration via OpenRouter
- Supabase backend with edge functions
- Interactive causal ripple analysis
- Policy pathway explorer with probability estimates

#### Technical
- React 18 + TypeScript + Vite
- Tailwind CSS + shadcn/ui components
- Three.js / react-globe.gl for 3D visualization
- D3.js for force-directed graphs
- Framer Motion animations
- Supabase (Database, Edge Functions, Auth)
